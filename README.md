# Virtual Pet Prototype

A virtual pet game prototype built with TypeScript.

This project is an experiment to build a digital pet inspired by classic virtual pets and game companions.

Inspirations include:
- Tamagotchi
- Pokémon
- Mega Man Battle Network PET devices

The goal is to explore:
- game logic architecture
- state-based character behavior
- custom drawn art assets

---

## Current Features

- Basic interaction system
- Click interaction increases pet happiness

More systems will be added incrementally.

---

## Tech Stack

- TypeScript
- Vite

---

## Running the Project

Install dependencies:
``` 
npm install
```
Start the dev server:
``` 
npm run dev
```
